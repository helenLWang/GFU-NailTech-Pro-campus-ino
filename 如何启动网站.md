# 🔧 如何启动 GFU Nail 网站

## 问题诊断

如果浏览器访问失败，请按以下步骤操作：

### 步骤 1: 确认 Flask 已安装

打开命令提示符（CMD）或 PowerShell，在项目目录下运行：

```bash
pip install -r requirements.txt
```

### 步骤 2: 启动服务器

**方法 1 - 使用诊断脚本（推荐）：**
```bash
python fix_and_run.py
```

**方法 2 - 直接运行：**
```bash
python app.py
```

**方法 3 - Windows 批处理：**
```bash
start.bat
```

### 步骤 3: 查看启动信息

服务器启动后，你应该看到类似这样的信息：

```
 * Running on http://127.0.0.1:5000
 * Debug mode: on
```

### 步骤 4: 访问网站

在浏览器中打开以下任一地址：

- http://localhost:5000
- http://127.0.0.1:5000

## 常见问题解决

### ❌ 问题 1: "ModuleNotFoundError: No module named 'flask'"

**解决方法：**
```bash
pip install flask
```
或者
```bash
pip install -r requirements.txt
```

### ❌ 问题 2: 端口 5000 已被占用

**解决方法：**

1. 关闭占用端口的程序
2. 或者修改 `app.py` 最后一行，改为其他端口：
   ```python
   app.run(debug=True, host='0.0.0.0', port=5001)  # 改为 5001
   ```

### ❌ 问题 3: 浏览器显示 "无法访问此网站"

**检查清单：**

1. ✅ 确认服务器已启动（看到 "Running on..." 信息）
2. ✅ 确认使用的是 `http://` 而不是 `https://`
3. ✅ 确认端口号正确（默认是 5000）
4. ✅ 尝试使用 `127.0.0.1` 而不是 `localhost`
5. ✅ 检查防火墙是否阻止了连接

### ❌ 问题 4: 看到错误信息

**如果看到导入错误：**

运行诊断脚本：
```bash
python fix_and_run.py
```

这个脚本会自动检查并修复常见问题。

## 快速测试

运行这个命令测试 Flask 是否正常工作：

```bash
python -c "import flask; print('Flask version:', flask.__version__)"
```

如果显示版本号，说明 Flask 安装成功。

## 需要帮助？

1. 运行 `python fix_and_run.py` 查看详细错误信息
2. 检查命令提示符中的错误消息
3. 确认你在正确的目录（`C:\Users\13360\Desktop\dist`）
